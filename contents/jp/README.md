#### 📦 実行方法

開発環境バージョン  
node.js -v12.18.4  
npm -v 6.14.6  
Visual Studio code -v 1.57  
git -v 2.32.0.window.1

```sh
$ npm install npx -g
```

```sh
$ npx create-react-app movie_app_2021
```

```sh
$ npm start
```

```sh
$ npm i prop-types
```

```sh
$ npm i axios
```

```sh
$ npm i gh-pages
```

```sh
$ npm run build   <<json.scripts
```

```sh
$ npm run deploy   <<json.scripts
```
